# Command Catalogue

- `nova chat`: local Ollama/fallback chat with memory context.
- `nova think`: builds a plan without action.
- `nova run`: executes a plan in dry-run unless confirmation is provided.
- `nova memory`: local SQLite memory CRUD/export.
- `nova task`: local task manager.
- `nova index` / `nova ask`: document RAG with source chunks.
- `nova study`: study notes and flashcards from indexed docs.
- `nova scan` / `nova cleanup-plan` / `nova search` / `nova duplicates`: file intelligence.
- `nova data profile`: dataset profiler and report.
- `nova code analyze` / `nova code security-review`: code intelligence.
- `nova forge`: project blueprint and safe builder.
- `nova plugins`: permissioned built-in plugin registry.
- `nova workflows`: recipe-based multi-step workflows.
- `nova dashboard`: local Streamlit dashboard.
- `nova api`: local FastAPI entry point instructions.
