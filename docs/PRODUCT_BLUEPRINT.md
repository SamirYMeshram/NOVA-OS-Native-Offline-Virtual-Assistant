# Product Blueprint

NOVA Sovereign AI is a local-first personal AI operating layer. It combines local model orchestration, memory, document intelligence, file intelligence, safe automation, data analysis, coding assistance, dashboard UX, and plugins.

## Product layers

1. Interface layer: CLI, dashboard, local API, future voice.
2. Intelligence layer: command router, planner, local model manager, retrieval.
3. Tool layer: memory, RAG, file intelligence, data analysis, code tools, automation.
4. Safety layer: path protection, permission gates, audit logs, redaction, confirmation workflows.
5. Persistence layer: SQLite memory, document chunks, audit JSONL, undo plans.
6. Expansion layer: plugins and capability catalog.
