# Offline Setup

Internet is only needed for initial installation and model downloads.

1. Install Python 3.11+.
2. Install NOVA dependencies.
3. Install Ollama if you want local AI model responses.
4. Pull local models before going offline.
5. Run NOVA normally without internet.

Core offline features after setup:

- Memory
- Tasks
- File scan and organization planning
- Document indexing/search
- Dataset profiling
- Dashboard
- Code project generation
- System monitor
- Automation safety checks
