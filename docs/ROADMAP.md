# Roadmap

This is a strong local foundation, not a finished multi-year JARVIS. Next milestones:

- Persistent file watcher with incremental index updates.
- Full local TTS/STT adapters with whisper.cpp / Piper detection.
- Richer Streamlit dashboard widgets and WebSocket chat streaming.
- Dedicated llama.cpp backend.
- Real permissions UI for plugins.
- Local workflow recorder/replayer.
- Better OCR through optional Tesseract/easyocr adapters.
- Model benchmark page.
- Per-folder policy profiles.
- Sync-free encrypted memory export.
